# Project Backlog: Crypto ML Predictor

This document outlines the high-level milestones and phases for the cryptocurrency price prediction project, ensuring alignment with the Medallion architecture and Senior ML standards.

---

## Phase 1: Environment and Data Infrastructure
*   Milestone: Successful connection to BigQuery Gold datasets and local environment configuration.
*   Objective: Establish a secure and reproducible data ingestion pipeline.

## Phase 2: Statistical Hypothesis Validation
*   Milestone: Quantitative confirmation of Volume Shock correlation with price returns.
*   Objective: Validate the mathematical basis of the prediction model before implementation.

## Phase 3: Baseline Model Development
*   Milestone: Trained XGBoost model with experiment tracking via MLflow.
*   Objective: Create a performance benchmark using time-series cross-validation.

## Phase 4: Production Refactoring and Observability
*   Milestone: Modularization of technical features and deployment-ready inference scripts.
*   Objective: Transition from exploratory notebooks to a professional, maintainable codebase.
